# Guizang Skill 商业授权合作说明

这份说明适用于希望在自己的 AI 产品、Agent 平台、创作工具、插件市场或工作流系统中接入 Guizang Skill 能力的合作方。

Guizang Skill 系列的核心能力包括但不限于：根据正文语义自动规划图文卡片、调用真实图库素材、处理截图与网页素材、生成地图/路线组件、完成多比例社交图片排版、生成高质量 PPT/演示文稿，以及对中文 AI 文本进行自然化改写。

本说明中的价格为首批合作参考价，具体合作范围、产品线、授权期限、适配深度、结算方式和交付内容，以双方最终签署的协议为准。

## 热门项目介绍

目前较受欢迎、适合商业授权或平台合作的项目主要包括以下三个。

### 1. Guizang Social Card Skill

项目地址：[op7418/guizang-social-card-skill](https://github.com/op7418/guizang-social-card-skill)

这个 Skill 面向小红书、小绿书、微信公众号、视频封面、社交媒体图文卡片等场景。它可以根据正文内容自动拆解页面结构，结合真实图片、截图、地图、商品图或网页素材，生成多比例、多页面的社交卡片。

适合接入的产品场景：

- AI 写作工具中的一键配图能力。
- 小红书 / 小绿书 / 公众号内容创作工具。
- Agent 平台中的图文卡片生成 Skill。
- 视频工具中的封面生成能力。
- 企业内容营销、产品说明、教程卡片生成场景。

核心价值：

- 减少用户手工找图、裁图、排版的成本。
- 避免纯色块、emoji 或 AI 插画导致的廉价感。
- 支持真实图库、截图、地图等证据型视觉素材。
- 输出可以直接用于社交平台发布的成品图。

### 2. Guizang PPT Skill

项目地址：[op7418/guizang-ppt-skill](https://github.com/op7418/guizang-ppt-skill)

这个 Skill 面向 PPT、演示文稿、汇报材料、课程课件、商业提案和视觉化表达场景。它能够根据用户提供的主题、文案、资料或结构，生成具有明确视觉风格的演示页面。

适合接入的产品场景：

- AI PPT 生成工具。
- 办公套件、在线文档、知识库和协作平台。
- 企业汇报、教育培训、公开演讲、课程内容生成。
- Agent 平台中的 PPT 生成或美化能力。
- 将文章、文档、会议纪要转换为演示稿的工作流。

核心价值：

- 帮用户从空白页直接进入可用的视觉表达。
- 提供比普通模板更强的排版、风格和叙事能力。
- 适合和文档、知识库、会议纪要、商业分析等工具结合。
- 可以作为 AI 办公产品的重要差异化能力。

### 3. Humanizer-zh

项目地址：[op7418/Humanizer-zh](https://github.com/op7418/Humanizer-zh)

这个 Skill 面向中文文本自然化、去除 AI 味、内容润色和表达优化场景。它可以识别并修复常见的 AI 写作痕迹，让中文内容更自然、更像真人书写。

适合接入的产品场景：

- AI 写作工具中的“一键去 AI 味”功能。
- 内容平台、公众号、小红书、知乎、视频脚本等创作工具。
- 简历、邮件、商业文案、社媒文案润色。
- 教育、办公、营销、知识管理类产品。
- Agent 平台中的中文文本优化 Skill。

核心价值：

- 提升 AI 生成文本的可读性和真实感。
- 降低用户后期人工润色成本。
- 适合放在生成链路的最后一步，作为文本质检和风格修正能力。
- 对中文场景更友好，不只是简单改写或同义词替换。

以上三个项目可以单独授权，也可以组合授权。组合授权适合希望一次性补齐“写作、配图、PPT、润色”完整内容生产链路的平台。

## 合作等级

### 1. 深度内置授权

这是最高级别的合作方式，适合希望将 Guizang Skill 的能力深度集成到自有产品中的团队。

合作方可以将相关能力内置到自己的产品流程中，也可以选择不在产品界面中展示 Guizang Skill 的署名。经双方确认后，合作方可以对外将该能力作为自身产品能力的一部分进行介绍。

我会配合合作方完成必要的产品适配、效果优化、提示词与工作流调整，使该能力在合作方产品中达到较好的使用体验。必要时，我也可以配合合作方进行联合宣传或案例发布。

参考费用：

- 创业公司、早期团队或熟人合作：一次性授权费用为 10 万元起。
- 日活 10 万级以上产品：一次性授权费用为 30 万元起。

默认包含：

- 授权一个明确产品或一个明确产品线使用。
- 允许深度内置到合作方产品中。
- 允许不展示 Guizang Skill 署名。
- 提供基础接入建议和一定范围内的效果优化配合。
- 可根据实际情况配合对外宣传。

默认不包含：

- 源代码或知识产权转让。
- 独家授权或品类排他。
- 跨多个产品线、多个品牌或多个客户的转授权。
- 长期驻场开发、持续定制开发或无限期维护。
- 将 Guizang Skill 能力包装后再出售给第三方平台。

如果合作方需要独家授权、区域排他、品类排他、源代码买断、长期深度定制、白标转售或多产品线使用，需要单独报价和签署补充协议。

### 2. 上架与露出合作

这是标准合作方式，适合希望在自己的平台中上架 Guizang Skill，但不需要深度定制或买断能力的合作方。

合作方可以在自己的平台、插件市场、Agent 市场或工作流市场中上架 Guizang Skill，并允许用户通过平台调用相关能力。我会保留 Guizang Skill 的版权和署名，不将其视为合作方自研能力。

参考费用：

- 固定合作费用：3 万元。

这笔费用也可以视为对 Guizang Skill 项目的赞助。作为回报，我会在项目 README、Skill 文档或相关说明中露出合作方品牌，具体露出位置和方式由双方确认。

默认包含：

- 授权合作方在一个平台或一个插件市场中上架 Guizang Skill。
- 保留 Guizang Skill 名称、版权和署名。
- 提供基础上架支持。
- 提供基础版本更新，但不承诺深度适配。
- 在 Guizang Skill 项目文档中露出合作方品牌。

默认不包含：

- 去署名或白标使用。
- 宣称该能力为合作方自研。
- 深度产品定制、私有化重构或长期专项维护。
- 独家授权、排他授权或转授权。

### 3. 收益分成模式

这是最低门槛的合作方式，适合已经具备付费能力的平台，例如用户可以通过积分、Token、额度包、订阅会员或按次付费来使用 Skills 的平台。

合作方可以将 Guizang Skill 上架到平台中，由用户按平台规则付费使用。合作方需要定期向我结算由 Guizang Skill 产生的收益分成。

参考方式：

- 不收取固定上架费用，或仅收取很低的接入费用。
- 合作方按月或按季度结算收益分成。
- 分成比例根据平台计费方式、流量规模、推广资源和技术支持成本另行确认。

默认包含：

- 授权合作方在平台中上架 Guizang Skill。
- 用户可以通过平台的积分、Token、额度或付费系统调用该能力。
- 我不提供额外深度适配，只维护通用版本能力。

默认要求：

- 合作方需要提供清晰的调用数据、收入数据和结算周期。
- 合作方不得隐藏、混淆或挪用 Guizang Skill 产生的实际收益。
- 合作方需要在合理位置保留 Guizang Skill 名称或来源说明。
- 如果后续需要深度适配、去署名、白标或对外包装为自研能力，应升级为深度内置授权。

## 通用授权边界

除非双方另有书面约定，以上任一合作等级均不代表：

- 转让 Guizang Skill 的版权、商标、名称、源代码或其他知识产权。
- 授予合作方独家使用权。
- 允许合作方将 Guizang Skill 再授权、转售或封装给第三方。
- 允许合作方使用 Guizang Skill 的内容、代码、提示词、工作流或生成结果训练竞品模型或复刻类似产品。
- 允许合作方移除版权信息后，将 Guizang Skill 作为完全自有资产再次商业化。

如需上述权利，需要单独协商并签署独立协议。

## 合作流程

1. 合作方说明产品形态、接入场景、预计用户规模和商业模式。
2. 双方确认合作等级：深度内置授权、上架与露出合作，或收益分成模式。
3. 双方确认授权范围，包括产品线、平台、使用方式、是否署名、是否需要适配、是否需要宣传配合。
4. 双方确认费用、付款方式、结算周期和交付内容。
5. 签署协议并完成付款或结算配置。
6. 我提供对应等级的接入、上架或适配支持。
